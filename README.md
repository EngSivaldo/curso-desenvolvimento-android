# Curso de Desenvolvimento Android Nativo 🚀

Repositório centralizado para armazenar todos os projetos, conceitos práticos e exercícios desenvolvidos ao longo das aulas de Engenharia de Software.

## 📂 Estrutura de Estudos

### 📱 [01 - Calculadora Álcool ou Gasolina](./01-AulaAlcoolOuGasolina)
* **Conceitos aprendidos:** Manipulação básica de layouts (XML), validação de campos numéricos de entrada (`EditText`), cálculo de viabilidade de combustível e exibição de resultados na tela.
* **Componentes:** `TextInputLayout`, `Button`, `TextView`.

### 🔀 [02 - Ciclo de Vida e Navegação com Activities](./02-AulaActivityFragment)
* **Conceitos aprendidos:** Transição entre telas utilizando a classe `Intent`, boas práticas para evitar a duplicação de telas na pilha de execução do Android usando as flags `CLEAR_TOP` e `SINGLE_TOP`.
* **Fluxo de dados:** Envio seguro de dados estruturados chave-valor entre componentes com `putExtra` e tratamento defensivo de dados nulos no recebimento.
* **Componentes:** `MaterialCardView`, `MaterialButton`, `ConstraintLayout`.
